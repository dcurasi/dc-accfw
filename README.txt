== Description ==

Advanced Coupon Conditions for Woocommerce allows you to create coupons with advanced attributes.
It is based on the [WordPress Plugin Boilerplate] (https://github.com/DevinVinson/WordPress-Plugin-Boilerplate) and it is tested with Woocommerce 2.4+.

= Features include =

* Create coupons based on payment methods
* Create coupons based on billing countries
* Create coupons based on shipping countries
* Create coupons based on postal codes (you can include or exclude)


= Available Languages =

* English - UNITED KINGDOM


== Installation ==

1. Unzip the downloaded zip file.
2. Upload `woocommerce-advanced-coupon-conditions` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Go to Woocommerce > Coupon codes and create new coupon
